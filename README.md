# 🌐 Web Development Journey – Week 1

This repository documents my **Week 1 progress** in learning **Web Development**.
It includes hands-on practice with **HTML, CSS, and JavaScript**, along with mini-projects and concept-based exercises.

---

## 🚀 Overview

In this week, I focused on:

* Building a strong foundation in **HTML structure**
* Styling using **CSS & Flexbox/Grid**
* Creating **responsive layouts**
* Implementing **forms with validation**
* Writing basic **JavaScript with DOM manipulation**

---

## 📅 Week 1 Roadmap

### 🟢 Day 1 – HTML Basics

**Task:** Create a Personal Portfolio Page

* Sections: About, Projects, Contact
* Navigation bar using `<nav>` and anchor tags
* Semantic HTML structure
* Internal linking between sections

**Concepts Covered:**

* HTML Tags
* Semantic HTML
* Page Structure

---

### 🎨 Day 1 – CSS Styling

**Task:** Style the Portfolio Page

* Use Flexbox for layout
* Apply fonts, colors, spacing
* Add hover effects & transitions
* Use Google Fonts

**Concepts Covered:**

* CSS Selectors
* Flexbox
* Colors & Typography

---

### 📝 Day 2 – Forms + Validation

**Task:** Create a Contact Form

* Fields: Name, Email, Message
* HTML5 validation using `required`
* Email input type & patterns
* Use `fieldset` and `legend`

**Concepts Covered:**

* Form Elements
* Input Types
* HTML Validation

---

### 📐 Day 3 – CSS Layouts

**Task:** Build a 3-Column Responsive Layout

* Header, Main Content, Sidebar
* Use Flexbox + Grid
* Add media queries for responsiveness

**Concepts Covered:**

* Flexbox vs Grid
* Responsive Design
* Media Queries

---

### 🖼️ Day 4 – Page Design Implementation

**Task:** Recreate a Design Layout

* Follow a reference UI design
* Customize fonts, images, and colors
* Focus on visual design & layout accuracy

---

### ⚡ Day 5 – JavaScript Basics + DOM

**Task:** Build a Counter App

* Increment & Decrement buttons
* DOM manipulation using JS
* Use closures for state management
* Add min/max limits

**Concepts Covered:**

* Variables & Functions
* Event Handling
* DOM Manipulation
* Closures & Hoisting



## 💡 Key Learnings

* Writing clean and semantic HTML
* Creating responsive layouts using Flexbox & Grid
* Styling with consistency and design principles
* Handling user input with forms and validation
* Using JavaScript to interact with the DOM
